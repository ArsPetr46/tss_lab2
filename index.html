WEB INFORMATION SYSTEM FOR EXCHANGING MESSAGES BETWEEN USERS USING JAVA STACK

  • По-перше, важливо зазначити актуальність теми сервісу для обміну повідомленнями між користувачами через інтернет. Наразі цей процес, ймовірно, є найбільш популярною потребою серед всіх користувачів. Для ефективного обміну повідомленнями в реальному часі необхідно створити гнучку та ефективну серверну частину додатка, яка буде здатна не тільки вчасно доставляти повідомлення до необхідних користувачів, а й виконувати інші базові операції над ними (видалення, зміна). Важливою частиною серверної частини є ефективне проектування бази даних та правильний вибір системи для її управління.
  • По-друге, в темі зазначено, що клієнтська частина буде мати вигляд вебдодатка. Уніфікація браузерів призвела до того, що при розробці достатньо створити одну версію додатка, і доступ до неї будуть мати всі користувачі, попри тип їх пристрою та операційну систему. Використання браузера для надання доступу до додатка є оптимальним рішенням, яке гарантує уніфікацію версій додатка, значне полегшення процесу впровадження оновлень та отримання доступу до самого додатку.
